# 测试数据

本目录存放各轮测试 / 评估的运行数据快照，按版本子目录隔离，互不干扰。

## 约定

- 每轮测试数据放在独立的 `vN/` 子目录下（如 `v1/`、`v2/`），版本间互不覆盖。
- 子目录内容通常包含：会话运行产物快照（runs）、评估打分、oracle、coverage-ledger、evidence 等。
- 旧版保留供回归对比，不删除。

## 边界

- 测试数据仅用于**离线回归与覆盖率度量**，不进入攻击 prompt（参见 `design/评估方案/` 与 `engine/benchmark_eval.py`）。
- 含敏感信息（cookie、凭证、授权文本）的运行数据**不应入库**；入库前必须脱敏。
